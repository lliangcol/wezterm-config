# WezTerm 快捷键图片生成指南

本目录用于存放生成的快捷键速查图。

## 快速开始（5 分钟）

### 步骤 1：复制下面的内容

选择以下任一版本复制（推荐从"精简版"开始）：

---

### 版本 A：精简版（最常用）

```
WezTerm 快捷键速查 - 常用功能

修饰键说明
  macOS:         SUPER = Command | SUPER_REV = Cmd+Ctrl
  Windows/Linux: SUPER = Alt     | SUPER_REV = Alt+Ctrl
  Leader:        Space + SUPER_REV

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

常用功能
  F1              进入复制模式
  F2              打开命令面板
  F3              打开启动器
  F11             切换全屏
  SUPER + f       搜索（不区分大小写）
  SUPER_REV + u   快速打开 URL

标签页
  SUPER + t       新建标签页
  SUPER_REV + w   关闭当前标签页
  SUPER + [       切换到左侧标签页
  SUPER + ]       切换到右侧标签页
  SUPER_REV + [   向左移动标签页
  SUPER_REV + ]   向右移动标签页

分屏（Pane）
  SUPER + \       垂直分屏
  SUPER_REV + \   水平分屏
  SUPER + w       关闭当前分屏
  SUPER + Enter   缩放当前分屏
  SUPER_REV + k   切换到上方分屏
  SUPER_REV + j   切换到下方分屏
  SUPER_REV + h   切换到左侧分屏
  SUPER_REV + l   切换到右侧分屏

窗口
  SUPER + n       新建窗口
  SUPER + -       缩小窗口
  SUPER + =       放大窗口
  SUPER_REV + ⏎   最大化窗口

复制/粘贴
  Ctrl+Shift+c    复制
  Ctrl+Shift+v    粘贴
```

## 步骤 2：访问在线工具

选择以下任一工具（推荐 Carbon）：
- **Carbon**: https://carbon.now.sh
- **Ray.so**: https://ray.so

## 步骤 3：配置样式（推荐设置）

粘贴内容后，在工具中配置：

**极客风格主题推荐：**
- **Dracula**: 经典暗色，紫粉青配色（推荐）
- **Monokai**: 低对比度，舒适护眼
- **Synthwave '84**: 赛博朋克荧光风格（最炫酷）
- **Night Owl**: 现代暗色，高可读性

**字体推荐：**
- Fira Code（推荐，支持连字）
- JetBrains Mono（清晰现代）
- Hack（紧凑高效）

**其他设置：**
- **语言**: Text 或 Markdown
- **背景**: 开启（暗色背景）
- **窗口装饰**: 可选（macOS 风格装饰）
- **导出**: 2x 或 4x 分辨率（高清）

## 步骤 4：导出图片

1. 调整窗口大小以适配内容
2. 点击 "Export" 导出 PNG
3. 保存为：
   - 精简版: `wezterm-cheatsheet.png`
   - 完整版: `wezterm-cheatsheet-full.png`

## 截图预览

完成后将图片保存到本目录：
- `docs/images/wezterm-cheatsheet.png` - 精简版（推荐日常使用）
- `docs/images/wezterm-cheatsheet-full.png` - 完整版（参考文档）

## 样式参考

**Carbon 推荐配置（复制粘贴）：**
```
Theme: Dracula
Font: Fira Code
Font Size: 14px
Line Height: 133%
Background: Yes
Window Controls: Yes
Drop Shadow: Yes
```

**Ray.so 推荐配置：**
```
Theme: Breeze
Padding: 64
Language: Text
Background: Yes
Dark Mode: Yes
```
